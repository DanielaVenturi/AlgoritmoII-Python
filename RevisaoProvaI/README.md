


### 1. O que é CPU?
A **CPU (Unidade Central de Processamento)** é o cérebro do computador.  
Ela executa instruções, processa dados e controla todas as operações do sistema.

---

### 2. Qual a função da memória RAM?
A **memória RAM** armazena temporariamente os dados e programas em uso.  
Ela é rápida, porém **volátil** (os dados são apagados ao desligar o computador).

---

### 3. Diferença entre armazenamento (HD/SSD) e memória RAM

- **RAM:** rápida e temporária  
- **HD/SSD:** armazenamento permanente  

---

### 4. Dispositivos de entrada

São usados para enviar dados ao computador.

- Teclado  
- Mouse  
- Microfone  
- Scanner  

---

### 5. Dispositivos de saída

Exibem as informações processadas pelo computador.

- Monitor  
- Impressora  
- Caixa de som  
- Fones de ouvido
